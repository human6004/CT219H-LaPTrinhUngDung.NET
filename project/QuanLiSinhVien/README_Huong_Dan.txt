Lưu ý:
- Trước khi giải nén trên Windows, nên Unblock file zip để tránh lỗi .resx bị chặn.
- Mở solution tại: src\QuanLySinhVien.sln


Cách mở project:
1. Giải nén ra đường dẫn ngắn, ví dụ D:\QLSV_Final
2. Không để trong OneDrive
3. Mở file Demo_QuanLySinhVien_C#\QuanLySinhVien.sln
4. Build/Rebuild Solution rồi chạy

Nếu SQL Server chưa có database QuanLySinhVien:
- Có thể để chương trình tự thử restore từ file DB\QuanLySinhVien.bak
- Hoặc restore thủ công trong SSMS
